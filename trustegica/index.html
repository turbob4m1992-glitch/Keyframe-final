Partnership Overview

02 / ENGINEERING

Why Custom Development

Our original proposal fulfills all technical and ownership requirements outlined in your research document. Fulfilling these standards requires significant custom engineering effort beyond standard website builds:

  • Custom Architecture vs. WordPress: Custom-coded from scratch for sub-second load times, maximum security, and 99+ PageSpeed performance (compared to the current site's mobile score of 34/100).
  • Dual-Language Engineering (EN + Arabic RTL): Separate layout engines, typography hierarchy, and directionality logic across every single corporate page.
  • Content & SEO Control Center CMS: A bespoke backend that lets non-technical users edit content, add pages, and manage SEO without touching code.
  • 100% IP & Source Code Handover: Un-locked source code ownership with zero developer lock-in.
To ensure budget considerations do not prevent us from moving forward, we have revised the one-time development price from USD 5,700 down to USD 4,900.
03 / SUPPORT

Complimentary 1-Year Maintenance

Because we trust what has been proposed and want to build a lasting relationship with Trustegic, we are including 1 Full Year of Technical Maintenance & Hosting Support at USD 0 / month (a USD 5,400 value).

  • Included Hours: Up to 4 hours per month of technical support.
  • Included Scope: Security vulnerability monitoring, server & software updates, SSL renewal, backups, bug fixes, and performance monitoring.
  • Overage Rate: Any technical work exceeding 4 hours in a month billed at USD 60 / hour.
  • Exclusions: Covers system stability and uptime (does NOT include new content creation, layout redesigns, or building new pages).
